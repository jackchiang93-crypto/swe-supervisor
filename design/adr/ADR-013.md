# ADR-013: Dashboard 靜態儀表板

狀態: accepted
日期: 2026-06-19

## 背景
overview 是文字版。要更一眼掌握,需要視覺化儀表板,但單人專案不該為此跑 server。

## 決策
新增 `supervisor/dashboard.py`:`render_dashboard()` 複用 `dossier.discover_specs`,
產生**自包含 HTML**(內嵌 CSS,無 CDN/外連)。CLI `dashboard --out` 寫檔,瀏覽器直開。
顯示總進度、每 SPEC 狀態色塊、未完成清單。

## 影響
- 新增 `supervisor/dashboard.py`、CLI `dashboard`。
- 取捨:靜態快照(非即時)——重跑命令才更新;但零依賴、可離線、可附進 PR。
- 不引入前端框架/伺服器,純字串模板。
